FOLDER MUZYKI - INSTRUKCJE

Ten folder zawiera pliki muzyczne, które będą automatycznie ładowane przez MusicService.

FORMATY PLIKÓW:
- .mp3 (zalecane)
- .ogg (zalecane)
- .wav
- .aiff

INSTRUKCJE:
1. Umieść pliki audio w tym folderze
2. Upewnij się, że pliki mają poprawny format
3. MusicService automatycznie załaduje wszystkie pliki przy starcie gry
4. Utwory będą losowo odtwarzane w pętli

UWAGI:
- Pliki są ładowane do pamięci przy starcie
- Używaj kompresji dla lepszej wydajności
- Rozważ rozmiar plików (zalecane < 10MB na utwór)
- Nazwy plików będą wyświetlane w logach

PRZYKŁADOWE NAZWY:
- background_music_01.mp3
- game_theme.ogg
- menu_music.wav
